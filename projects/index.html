Algal Diversity Analysis

October 2023-February 2024

• Collaborated on a Texas State University interdisciplinary project analyzing algal diversity using FlowCam data, enhancing understanding of environmental processes.

• Conducted advanced preprocessing on algal cell images using computer vision techniques such as thresholding, blob detection, and edge detection to eliminate various forms of noise, including empty, partial, and duplicate cells, preparing the dataset for enhanced analytical accuracy.

• Refined data analysis through unsupervised clustering; initial trials did not yield distinct separations, prompting the use of feature extraction with Hu Moments that significantly improved the clustering outcomes, leading to more meaningful groupings and insights for algal cell analysis.

• Leveraged insights from machine learning and computer vision for algal cell classification and counting, streamlining biological data analysis.

• Presented project insights and methodologies as a guest lecturer, sharing findings in a PhD-level Computer Vision class (CS 7323 at Texas State University).

SIGNIFY: ASL Learning Tool

January 2024-May 2024

• Developed SIGNIFY, an interactive web-based tool for learning American Sign Language (ASL) fingerspelling, designed to support users from beginners to intermediate levels through structured content and gameplay.

• Implemented template matching for recognizing static ASL signs, and utilized dynamic time warping for dynamic signs, accommodating the temporal aspects of ASL fingerspelling movements; templates are compared by normalizing coordinates and measuring similarity through Euclidean distance.

• Leveraged real-time webcam gesture recognition using Google MediaPipe’s hand detection pre-trained model to provide immediate feedback, enhancing the interactive learning experience within the application.

• Designed learning and gaming modes, allowing users to progress from basic ASL alphabet fingerspelling to more complex techniques including dynamic signs.

• Developed the application using plain JavaScript and styled with Bootstrap 5, leveraging Node.js and npm for robust dependency management and efficient operation of the development server.

MRI Brain Tumor Detection

December 2023

• Preprocessed a Kaggle MRI brain image dataset and dove into why deep learning, especially Convolutional Neural Networks (CNNs), work well for brain tumor detection on MRI images. There is a Jupyter Notebook titled "tumor-classification-cnn.ipynb" that walks through building and tuning a CNN model, showing how it's great for image classification, especially with medical images like MRIs

• The choice of deep learning was driven by its proven capability in handling complex image recognition tasks, and CNNs were selected due to their effectiveness in image classification problems, particularly in identifying patterns and features in medical imaging

• Following the development and fine-tuning of the CNN model in the notebook, this project extends to the realm of practical application through a web interface. Built with Flask, the web application leverages the trained CNN model to provide real-time predictions on pre-loaded MRI images (subset of the test set). This integration not only showcases the model's capabilities in a user-friendly format but also demonstrates the potential of deep learning models in real-world applications
